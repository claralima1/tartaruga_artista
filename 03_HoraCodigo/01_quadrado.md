# Quadrado

Bem-vindo à Tartaruga Artista!

Primeiro, vamos tentar fazer um quadrado simples usando os métodos
```turtle.right(90)``` (ou ```turtle.left(90)```) e ```turtle.forward(100)```.
Cada lado deve medir 100 pixels.
